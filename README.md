# MI-PORTAFOLIO-PROFESIONAL

📌 Objetivo del Proyecto
Este portafolio ha sido diseñado y desarrollado con el propósito de consolidar mi identidad digital como estudiante de Ingeniería de Sistemas. El objetivo principal es mostrar de forma pública mis habilidades técnicas, proyectos realizados, trayectoria académica y competencias en el desarrollo de software, sirviendo como una herramienta clave para mi inserción y crecimiento en el mercado laboral tecnológico.

 🛠️ Tecnologías Utilizadas

Para la construcción de este sitio web se emplearon tecnologías modernas de desarrollo frontend que garantizan un diseño limpio, adaptativo y profesional:
- **HTML5:** Estructuración semántica del contenido.
- **CSS3 / Bootstrap:** Estilos personalizados, grillas y componentes responsivos para visualización en móviles y computadoras.
- **JavaScript (Opcional):** Interactividad básica y dinamismo en la página.
- **Inteligencia Artificial:** Utilizada de manera estratégica para la optimización de copys, prompts de diseño y refinamiento de secciones de marca personal GEMINI.
